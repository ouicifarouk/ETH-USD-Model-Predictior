# ETH-USD-Model-Predictior
A supervised learning machine learning model of the linear regression type with real-time hourly data for the last 23 months updated hourly from yfinance was used to predict the highest and lowest price of ETH against USD The RMSE model had an average squared error of approximately 15 
The model is good for learning and using data, and for observing its accuracy, especially since it achieves an average squared error of 15 and a standard deviation of approximately 3.
